## Best Use
- Use hooks 1-5 for launch and attention
- Use hooks 6-14 for authority and trust
- Use hooks 15-20 for conversion and build-in-public follow-ups

## Hook Writing Rule
If the hook does not trigger one of these reactions, rewrite it:
- "That's true"
- "That is exactly my problem"
- "I want to know how"
